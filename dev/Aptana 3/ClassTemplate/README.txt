Pour installer le template de classe Mediboard :
		- Dans Aptana : Commands -> PHP -> Edit this bundle
		- Cela va créer un dossier "php.ruble" dans votre dossier personnel
		- Remplacer le fichier "templates.rb" dans le répertoire "php.ruble/templates"
		- Copier le fichier class_mediboard.php
		- Redémarrer Aptana
		- Pour l'utiliser, New From Template -> PHP -> Mediboard Class Template
