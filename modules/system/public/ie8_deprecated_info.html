Fin de prise en charge du navigateur Internet Explorer 8

Le navigateur Internet Explorer 8 ne sera plus pris en charge par Mediboard à partir de la version de juillet 2014 afin de se concentrer sur les versions plus récentes et pérennes.

Cet arrêt de prise en charge est lié à la fin de vie du système d'exploitation Windows XP programmée pour le 8 avril 2014, dont Microsoft ne fera plus de mises à jour, même de sécurité, ce qui peut s'avérer dangereux pour vos données.

Qu'est ce que cela implique ?

Vous utilisez Windows XP : il est fortement recommandé de passer à une version plus récente de Windows (par exemple Windows 7 ou Windows 8) ou de renouveler votre matériel.

Vous utilisez Windows Vista, Windows 7 : si la version 8 d'Internet Explorer est utilisée, il vous faut appliquer les mises à jour Windows jusqu'à l'installation de la version 9, 10 ou 11 du navigateur afin de profiter pleinement de Mediboard.

Une autre solution est d'installer un navigateur alternatif, comme Mozilla Firefox ou Google Chrome, qui sont eux pris en charge par Mediboard, et continuent d'être mis à jour sous Windows XP.

Vous pouvez contacter votre service informatique afin de procéder à ces manipulations.
